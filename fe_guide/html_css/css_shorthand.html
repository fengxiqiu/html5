.selector {
    color:#ee7733;
}

简写后 ：

.selector {
    color:#e73;     //用16进制书写时，如果每种颜色的值相同，就可以写成一个。如#113366 -- #136
}
margin
.selector {
    margin-top:1px;
    margin-right:2px;
    margin-bottom:3px;
    margin-left:4px;
}

简写后 ：

.selector {
    margin:1px 2px 3px 4px ; //对应 top right bottom left
}

如果取值有重复，可以合并对应的值，合并规则如下

.selector {
    margin:1px;              //4个值相同
    margin:1px 2px;          //上下边距都为1px，左右边距均为2px，等同于margin:1px 2px 1px 2px
    margin:1px 2px 3px;      //右边距和左边距相同，等同于margin:1px 2px 3px 2px;
}
padding
.selector {
    padding:1px 2px 3px 4px ;  //top right bottom left  简写规则同margin
}
border
.selector {
    border-width;1px;
    border-style:solid;
    border-color:#666;
}

简写后 ：

.selector {
    border:1px solid #666;      //按照width、style和color的顺序简写,style不能省略，width和color不写时按照默认样式
}

可以对每条边采用缩写

 .selector {
     border-top:4px solid #333;
     border-right:3px solid #666;
     border-bottom:3px solid #666;
     border-left:4px solid #333;
}

也可以对每个属性采用缩写

 .selector {
     border-width：1px 2px 3px;       //最多可用四个值，缩写规则类似盒子大小的缩写，下同
     border-style：solid dashed dotted groove;
     border-color:red blue white black;
}

.selector {

    background-repeat: repeat || repeat-x || repeat-y || no-repeat;

简写后 :

css2
.selector {
    background: red url("") top left no-repeat scroll;
}

css3
.selector {
    background: background-image || background-position/background-size ||
                background-repeat || background-attachment || background-clip ||
                background-origin || background-color
}
//注意：background-size 必须写在background-position后面，且用"/"隔开
        避免部分浏览器不兼容css3属性，最后将css3属性分开写，并加上各自内核前缀
font
.selector {
    font-style: normal || italic || oblique;
    font-variant:normal || small-caps;
    font-weight: normal || bold || bolder || || lighter || (100-900);
    font-size: (number+unit) || (xx-small - xx-large);
    line-height: normal || (number+unit);
    font-family:name;
}

font 的简写必须严谨小心，按照一定的规则及顺序，否则容易出错，规则如下

.selector {
    font:italic small-caps bold         34px/1.5      "Arial Black"
           |         |       |              |                |
         style    variant  weight  font-size/line-height  family
}
//style variant weihgt为可选属性，书写时必须前置，放在font-size和font-family前面
//font-size font-family为必选属性，任一属性不填写都会导致整行失效
//line-height为可选属性，书写时必须跟在font-size后面，且用"/"分隔
list-style
.selector {
    list-style-type:none || disc || circle || square ||
                            decimal || lower-alpha || upper-alpha || lower-roman || upper-roman
    list-style-position:  inside || outside || inherit
    list-style-image:  (url) || none || inherit
}

简写后

.selector {
    disc outside none;  //如果url存在，则图片的优先级比list-style-type高，会覆盖其样式
}
